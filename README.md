# 🏥 QR Code Based Health Record System

A web-based health record management system that uses 
QR codes to instantly retrieve patient medical history.

## 🚀 Features
- Role-based login (Admin, Doctor, Patient)
- QR code generation on patient registration
- View diagnosis, prescriptions and medical history
- Secure and structured patient data management

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS
- **Database:** Oracle SQL

## 📊 Database Design
5 interconnected tables:
- User, Patient, Doctor, Diagnosis, Prescription
- Foreign key constraints for data integrity
